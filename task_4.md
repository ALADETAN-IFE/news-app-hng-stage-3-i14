# Mobile Track Stage 4 Task

Hello, Mobile Ninjas! :ninja::skin-tone-4:

Welcome to Stage 4. You've mastered mobile development; now it's time to expand your skills beyond a single platform. This stage will test your ability to build a single codebase that runs seamlessly on mobile, desktop, and web, demonstrating true cross-platform development expertise.

## Task Description

You have built mobile applications in previous stages. Now, you are required to transform your Stage 2 or Stage 3 app into a truly cross-platform application using a single codebase that works on:

- **Mobile**: Android and iOS (your existing app)
- **Desktop**: macOS, Windows, or Linux
- **Web**: modern browsers such as Chrome, Firefox, Safari, and Edge

Note: The goal of this task is to leverage cross-platform frameworks to make your existing app run everywhere while maintaining user experiences across different platforms.

## Task Deliverables

You are expected to build an app that:

- Adapts to different device platforms (mobile, desktop, and web) with a single codebase
- Detects the platform, adapts accordingly, and implements layouts that respond to screen sizes, not the used platform alone
- Implements different input methods and navigations such as keyboard shortcuts, mouse hover states for desktop/web, tab navigation for keyboard users, sidebar, menu bar, gestures, and more
- Implements a shared data layer across platforms, platform storage mechanisms, and offline functionality everywhere

### Desktop Version Must Include

- Resizable windows with proper content adaptation
- Application menu: File, Edit, View, Help
- Keyboard shortcuts, minimum 5
- Right-click context menus where appropriate
