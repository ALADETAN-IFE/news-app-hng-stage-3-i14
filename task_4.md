Hello, Mobile Ninjas!! :ninja::skin-tone-4: 
Welcome to Stage 4! You've mastered mobile development; it's time to expand your skills beyond a single platform. This stage will test your ability to build a single codebase that runs seamlessly on mobile, desktop, and web, demonstrating true cross-platform development expertise.

Task Description:
You have built mobile applications in previous stages. Now, you are required to transform your Stage 2 or Stage 3 app into a truly cross-platform application using a single codebase that works on:

Mobile: Android & iOS (your existing app)
Desktop: macOS, Windows, or Linux
Web: modern browsers (Chrome, Firefox, Safari, Edge)


Note: The goal of this task is to leverage cross-platform frameworks to make your existing app run everywhere while maintaining user experiences across different platforms.

Task Deliverables
You are expected to build an app that:

Adapts to different device platforms (mobile, desktop and web) with a single codebase
Detects the platform, adapts accordingly and implement layouts that respond to screen sizes & not the used platform alone
Must implement different input methods and navigations (keyboard shortcuts, mouse hover states for desktop/web, tab navigation for keyboard users, sidebar, menu bar, gestures, etc.).
Should implement shared data layer across platforms, platform storage mechanisms and offline functionality everywhere
The desktop version must include:
Resizable windows with proper content adaptation
Application menu (File, Edit, View, Help)
Keyboard shortcuts (minimum 5)
Right-click context menus where appropriate

A LinkedIn or X (Twitter) documentation post, tagging the HNG Internship account, outlining:
The development process
Challenges encountered
The purpose of the task
How the task has improved (or will improve) your development process
